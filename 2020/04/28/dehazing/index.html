一些较新的去雾（dehazing）算法

Apr 28, 2020 in 图像处理

提到去雾算法，首先想到的便是 2009 年的暗通道去雾算法，以及后续的导向滤波，搜索博客和 Github 发现了一些较新的去雾算法，也算是尝试跟上学术界的脚步。

  • 2013年基于对比度增强的去雾算法
    http://mcl.korea.ac.kr/projects/dehazing/#userconsent#

  • 2016年非局部均值去雾算法
    https://github.com/danaberman/non-local-dehazing
